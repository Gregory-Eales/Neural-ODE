## Description   
This is an implementation of the neural ode paper using pytorch lightning

## How to run   
First, install dependencies   
```bash
# clone project   
git clone https://github.com/YourGithubName/Your-project-name   

# install project   
cd Neural-ODE
pip install -e .   
pip install -r requirements.txt
 ```   
 Next, navigate to src and run it.   
 ```bash
# module folder
cd src/    

# run module   
python run.py    
```
